# GOF2-DLC-Windows
Mods that will enable supernova,kaamo club and valkyrie dlc for the galaxy on fire 2 windows version game

# How to install?
Install the modding api [here](https://github.com/1337Skid/KaamoClubModApi), grab the latest release of the dlcs on the release tab and don't forget to extract the data archive located in the "valkyrie_dlc" in the root folder of the game or else you'll have buggy textures etc..

# Disclaimer
Supernova will be available once the modding api is mature enough
